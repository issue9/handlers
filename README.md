handlers [![Build Status](https://travis-ci.org/issue9/handlers.svg?branch=master)](https://travis-ci.org/issue9/handlers)

handlers是对http.Handler接口的一系列实现，提供了大部分实用的功能。


### 安装

```shell
go get github.com/issue9/handlers
```
